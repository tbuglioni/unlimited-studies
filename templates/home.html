Learning, a whole story

Learn ... We can learn things, lessons, languages,stories "anything and everything". Anyway everything that can enter in our brain.

But who has the time and the energy to learn whatever they want with 24 hour days: / ??

This is how I asked myself the question

" How can you learn more and doing as little as possible? "

AND after going from application to application to memorize I started to create my application..



BUT !! But, yes there is a but

IF the app will help you to learn:
  • knowledge quickly
  • knowledge for a long time
  • with the lowest possible effort to provide

    • It will not do:
  • make connections between the elements
  • provide the knowledge for you
  • learn for you

    • Finally, I do not recommend it for:
  • work on a know-how
  • work on a know-how-to-be

    Well and how does it work ??

    Idea

    The idea is simple, study something as many times as necessary to remember it.

    With a gradual spacing between 2 revisions of the same element.

    List of things to know where you are:
    • Total : number of notes
    • Todoo : number of notes recto or verso to study
    • Lvl average : level average of all notes
    • Today recap : number of notes studied today
    • Month recap : number of notes studied this month

    Structure

    1. Book: At the first level: the book, with the main concept to study (example: "English" )
    2. Chapter: then X chapters,to split the book in some parts(example: vocabulary, grammar)
    3. Notes: and finally, X notes on both sides (example: hello/bonjour in the chapter "vocabulary"). You will learn in both directions recto -> verso and verso -> recto (or just one direction if you want)

      4. Book->Chapter->Notes

    Spaced repetition

    If you know the element you go to the next level and you have to wait x days to learn it again

    But if you don't know the element then it goes back to the level 1

    • level: 1->2 = +1 day
    • level: 2->3 = +1 day
    • level: 3->4 = +1 day
    • level: 4->5 = +1 day
    • level: 5->6 = +3 days
    • level: 6->7 = +7 days
    • level: 7->8 = +20 days
    • level: 8->9 = +90 days
    • level: 9->10 = +182j
    • level: 10->10 = +364j

    Study session

    You can run the game with 4 levels of difficulty:

    1. 40/10 (50 notes in the session)
    2. 100/50 (150 notes in the session)
    3. 200/100 (300 notes in the session)
    4. 250/250 (500 notes in the session)

    First number: number of notes with the level lower than 6

    Second number: number of notes with the level higher than 5

    BE CAREFUL! the game is based on:

    1. book order
    2. chapter order
    exemple :
    • book 1 is studied before book 2
    • book 1/chapter 1 is studied before book 1/chapter 2
